Características principales

Todo lo que necesitas para gestionar tu dieta baja en purinas

Análisis Individual

Consulta cualquier alimento, ingrediente o plato. Recibe información detallada sobre contenido de purinas, nivel de riesgo e índice de seguridad.

Análisis de Cartas

Fotografía menús de restaurantes y obtén un análisis completo de todos los platos. Filtra y ordena por nivel de seguridad.

Sistema de Semáforo

Clasificación visual intuitiva: verde (seguro), amarillo (moderado), rojo (precaución). Índice de seguridad de 0 a 100 puntos.

Contexto Temporal

Recomendaciones de frecuencia de consumo: diario, semanal u ocasional. Planifica tu dieta de forma inteligente.

Alternativas Inteligentes

Para alimentos de riesgo medio o alto, recibe sugerencias de alternativas más seguras y nutritivas.

Privacidad Total

Tu clave API se almacena solo en tu dispositivo. No guardamos tus búsquedas ni fotos. Sin servidores propios.

Cómo funciona

Empieza a usar IAGota en minutos

1

Obtén tu API Key

Crea una cuenta gratuita en OpenRouter y genera tu clave de API. Es rápido y sin costo inicial.

2

Configura la App

Descarga IAGota, abre configuración y pega tu API key. Opcionalmente elige tu modelo de IA preferido.

3

Consulta Alimentos

Escribe el nombre de cualquier alimento o fotografía una carta de restaurante para obtener análisis completo.

4

Toma Decisiones

Usa la información para planificar tu dieta. Sistema de semáforo, alternativas y consejos de preparación.

Key Features

Everything you need to manage your low-purine diet

Individual Analysis

Query any food, ingredient, or dish. Receive detailed information about purine content, risk level, and safety index.

Menu Analysis

Photograph restaurant menus and get a complete analysis of all dishes. Filter and sort by safety level.

Traffic Light System

Intuitive visual classification: green (safe), yellow (moderate), red (caution). Safety index from 0 to 100 points.

Temporal Context

Consumption frequency recommendations: daily, weekly, or occasional. Plan your diet intelligently.

Smart Alternatives

For medium or high-risk foods, receive suggestions for safer and more nutritious alternatives.

Total Privacy

Your API key is stored only on your device. We don't save your searches or photos. No proprietary servers.

How It Works

Start using IAGota in minutes

1

Get Your API Key

Create a free OpenRouter account and generate your API key. It's quick and has no initial cost.

2

Configure the App

Download IAGota, open settings, and paste your API key. Optionally choose your preferred AI model.

3

Query Foods

Type the name of any food or photograph a restaurant menu to get a complete analysis.

4

Make Decisions

Use the information to plan your diet. Traffic light system, alternatives, and preparation tips.
